## Active redesign: Ancient Greek Atrium (Three.js)

The entire portfolio is being rebuilt as an immersive 3D ancient Greek atrium.
Visitor enters a marble hall with Corinthian columns, coffered ceiling, and project panels on the walls.

### The .glb scene (already built)
`public/models/atrium.glb` was generated by `build_atrium.py` (Blender 5.1 headless).
It contains:
- Room shell (floor, 4 walls, ceiling) ŌĆö 24m ├Ś 10m ├Ś 7m
- 8 Corinthian columns with fluted shafts (4 per side)
- Coffered ceiling (3├Ś8 recessed panel grid)
- 6 project panel frames + canvas planes (tagged with `panel_id` and `is_panel_canvas`)
- 6 statue plinths between columns
- 2 entablature beams (one per side wall)
- Camera + lights (supplemented by Three.js at runtime)

### Panel canvas objects
The GLB contains flat planes named `Panel_{id}_canvas` with custom properties:
```
userData.is_panel_canvas = true
userData.panel_id = 'ema' | 'artsy' | 'augusts' | 'academia' | 'yummy' | 'thessaloniki'
```
At runtime, `AtriumScene.vue` replaces their material with a `THREE.CanvasTexture`
built from the `buildPanelTexture()` function (project title, tags, screenshot, buttons).

## Three.js conventions used in this project

- **No `@tresjs`** ŌĆö raw Three.js only, instantiated in `onMounted`, cleaned up in `onBeforeUnmount`
- `GLTFLoader` from `three/examples/jsm/loaders/GLTFLoader`
- `OrbitControls` from `three/examples/jsm/controls/OrbitControls`
- `RGBELoader` from `three/examples/jsm/loaders/RGBELoader` (HDRI)
- All panel canvases are `THREE.CanvasTexture` built at runtime via the `buildPanelTexture()` function
- Hover detection via `THREE.Raycaster` in the animation loop
- Camera movement uses a manual lerp loop (no GSAP dependency)

## Agent instructions

When working on this repo:

1. **Never use `bpy.ops` in Blender scripts** ŌĆö Blender 5.x background mode requires pure `bpy.data` / `bmesh` API
2. **Never open the Blender GUI** ŌĆö always use `--background --python`
3. **Always clean up Three.js** in `onBeforeUnmount` ŌĆö cancel `requestAnimationFrame`, call `renderer.dispose()`
4. **Panel canvases are identified by `userData.is_panel_canvas === true`** ŌĆö traverse the GLB scene and check this flag
5. **Nuxt 3 + Vue 3 only** ŌĆö no Options API, no Vue 2 patterns
6. **Tailwind for any 2D UI** ŌĆö the modal, HUD, loading screen all use Tailwind classes
7. **TypeScript** ŌĆö all new `.ts` / `.vue` files should be typed
8. **Do not break existing pages** ŌĆö the 3D scene is only on `pages/index.vue`; other pages stay untouched
9. **Netlify static** ŌĆö no server-side routes, no API routes, everything is client-side

## Photorealism goals (updated May 2026)

The scene must feel like you are *inside* the room ŌĆö no visible exterior, no sky.

### Rendering pipeline
- `THREE.WebGLRenderer` with `ACESFilmicToneMapping`, exposure 1.4
- `EffectComposer` with these passes in order:
  1. `RenderPass`
  2. `SSAOPass` (kernelRadius 0.4, maxDistance 0.08) ŌĆö contact shadows
  3. `UnrealBloomPass` (strength 0.25, threshold 0.85) ŌĆö subtle warm glow
  4. `OutputPass` ŌĆö colour space correction
- `shadowMap.type = PCFSoftShadowMap`, shadow map 4096├Ś4096
- `envMapIntensity = 0.6` on all MeshStandardMaterial objects

### Lighting setup (interior only)
| Light | Type | Color | Intensity | Purpose |
|---|---|---|---|---|
| Sun shaft L | DirectionalLight | `#FFD070` | 3.5 | Main dramatic light from upper-left |
| Sun shaft R | DirectionalLight | `#FFE090` | 1.2 | Cross-light from right, fills columns |
| Sky bounce | HemisphereLight | `#D0E0FF` / `#8A7A50` | 0.5 | Soft ambient from ceiling |
| Panel torches | PointLight ├Ś6 | `#FFAA40` | 0.8, range 5 | Warm glow near each project panel |
| Deep fill | AmbientLight | `#604030` | 0.3 | Prevents pure-black shadows |

### Enclosed navigation rules
- `scene.background = new THREE.Color(0x0A0806)` ŌĆö solid dark, never a sky
- `scene.environment = envMap` (HDRI) ŌĆö reflections only, NOT background
- `scene.fog = new THREE.Fog(0x1A1208, 18, 28)` ŌĆö interior fog, hides far walls softly
- Camera hard bounds (clampPosition): x ┬▒4.0, y -10.5ŌåÆ10.5, z 0.8ŌåÆ2.4
- No OrbitControls ŌĆö first-person WASD + mouse drag only
- Pitch clamped to ┬▒0.35 rad (can't look through floor or ceiling)

### Navigation controls
- WASD / arrow keys: walk forward/back/strafe
- Mouse drag: look around
- Touch drag: mobile look
- Click panel: open project modal
- ESC / click outside modal: close modal
